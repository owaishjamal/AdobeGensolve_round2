CURVETOPIA: A Journey into the World of Curves

Welcome

Welcome to Curvetopia, where we bring order and beauty to the world of 2D curves! Our mission is to identify, regularize, and beautify curves in 2D Euclidean space. We start by focusing on closed curves and progressively work with more complex shapes, covering symmetry and curve completion techniques.

Objective

Our objective is to take a raster image of line art and output a set of curves defined as a connected sequence of cubic Bezier curves. Initially, we simplify the input to polylines, which are sequences of points.

Problem Description

The problem is divided into three main sections:

  1. Regularize Curves: Identify regular shapes like straight lines, circles, ellipses, rectangles, and polygons from a set of curves.
  2. Explore Symmetry: Identify the presence of symmetry in closed shapes, beginning with reflection symmetries.
  3. Complete Incomplete Curves: Create algorithms to complete 2D curves that have gaps due to occlusion removal.

How to Run the Solution

Follow these steps to run the solution for each of the problems described above:

1. Regularize Curves

This script identifies and classifies geometric shapes such as straight lines, circles, ellipses, rectangles, and polygons.

python scripts/regularization.py

2. Explore Symmetry

This script analyzes the shapes for reflective and rotational symmetries.

python scripts/symmetry.py

3. Complete Incomplete Curves

This script uses interpolation techniques to complete 2D curves that have gaps due to occlusion or removal.

python scripts/fullcurve.py

4. Shape Recognition

This script trains a deep learning model on the provided shapes and then uses the trained model to recognize new shapes.

python scripts/trainmodel.py
python scripts/employmodel.py
